El Interés simple (I) es el interés que se cobra sobre el capital o valor presente (VP) que se presta por un periodo concreto.

Valor Futuro (VF) es el valor que tendrá una inversión en el futuro, considerando una tasa de interés o rendimiento. La fórmula para calcular el valor futuro es:  (VF): VP * (1 + (I * T))

El Valor Presente (VP) es el valor actual de una cantidad de dinero que se va a recibir en el futuro, descontado a una tasa de interés o rendimiento.(Capital), La fórmula para calcular el valor presente es: (VP): VF / (1 + (I * T))

El Porcentaje de Interés (i) es el costo de los fondos prestados, generalmente expresado como un porcentaje. La fórmula para calcular el porcentaje de interés es: (i): (VF / VP - 1) / T

El Total de tiempo (T) es la cantidad de tiempo en la que se desea realizar la operación. La fórmula para calcular el total de tiempo es: (T): (VF / VP - 1) / I
